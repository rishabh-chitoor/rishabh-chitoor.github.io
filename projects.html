Under Prof. Nitin Chandrachoodan
Github Repo

Novel 4-stage Cache Coherence Protocol Spring 2021

Towards B.Tech Thesis, under Prof. Madhu Mutyam
Github Repo | Thesis
  • Ideated a novel 4-state Cache Coherence Protocol designed to reduce the number of accesses to Main Memory: validated through self-designed coherence simulator & GEM5.
  • Theoretically proven to achieve lowest number of Main Memory accesses among all protocols.
